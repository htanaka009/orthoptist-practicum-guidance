1. 点をつなげる

1週間の場面を並べ、同じ課題や繰り返し出てきた所見を見つけます。

2. 検査の目的を言う

検査名だけでなく、何を確認するための検査かを書きます。

3. 来週へつなげる

来週見ること、実技で意識すること、調べ直すことを1〜2個に絞ります。

学生用：まずは必須4項目

週のまとめは、この4つが書ければ提出用の骨子になります。

必須1. 今週見た場面

症状・所見・検査場面を1つ選びます。広げすぎないことが大切です。

必須2. 関係しそうな原因

疾患・病態の候補を1〜2個に絞ります。確定ではなく、考える候補として書きます。

必須3. 必要な検査と見ること

原因を分けて考えるために、どの検査で何を見るかを書きます。

必須4. 来週確認したいこと

観察目標または実技目標を1〜2個に絞ります。

学生用：余裕があれば追加する項目

必須4項目が書けたあとに、必要に応じて追加します。

任意1. 今週分かったこと

単なる感想ではなく、検査と原因がどうつながったかを書きます。

任意2. まだ分からないこと

分からないことを、来週確認できる形にします。

任意3. 指導者に確認したいこと

質問は広げすぎず、今週見た場面に結びつけます。

学生用：調べることリストを整理する

疑問を全部レポートに入れる必要はありません。今週のまとめに使うものだけを選び、残りは保留リストに置きます。

保留リストの扱い：保留リストは未完了課題ではありません。今は深追いしないが、あとで確認する疑問の置き場です。保留にできることも、学習の整理です。

今週中に調べること：1〜2つ

週のまとめに関係する問いを選びます。疾患名や検査名だけでなく、場面と確認したいことを入れます。

保留リスト

今は深追いしないが、あとで確認したい疑問を書きます。数が多くてもかまいません。

学生用：本文にするときの短い型

型：今週見た場面 → 関係しそうな原因 → 必要な検査 → 来週確認したいこと

短い書き出し例

今週は、＿＿の場面を見た。この場面では、＿＿が関係しそうだと考えた。そのため、＿＿の検査で＿＿を確認する必要がある。来週は、＿＿と＿＿の結果がどうつながるかを見たい。

少し詳しく書く場合の型

今週は、＿＿の場面を複数回見た。最初は＿＿の検査手順に注目していたが、指導を受ける中で、＿＿を確認するために＿＿の検査が必要になることを学んだ。

特に、＿＿の検査では＿＿を見ることで、＿＿と＿＿を分けて考える手がかりになると分かった。来週は、＿＿の場面で、＿＿と＿＿の検査結果がどのようにつながっているかを意識して観察したい。

学生用：中級編で使いやすい接続表現

箇条書きの内容を本文にするときは、次の表現を使うと、場面・原因・検査・次週の課題がつながりやすくなります。

つなぐ内容使いやすい表現例
場面を述べる今週は、＿＿の場面を見た。今週は、視力が出にくい場面を見た。
原因へつなげるこの場面では、＿＿が関係しそうだと考えた。この場面では、屈折異常や弱視が関係しそうだと考えた。
検査へつなげるそのため、＿＿を確認するために＿＿検査が必要になる。そのため、矯正で改善するかを確認するために矯正視力が必要になる。
結果同士をつなげる＿＿だけでなく、＿＿と合わせて見る必要がある。視力だけでなく、屈折や眼位の結果と合わせて見る必要がある。
次週へつなげる来週は、＿＿と＿＿の結果がどうつながるかを見たい。来週は、屈折と矯正視力の結果がどうつながるかを見たい。

参考：テーマ別の例

弱視疑い

見る流れ：視力 → 屈折 → 矯正視力 → 眼位 → 両眼視 → 眼底確認

まとめ方：視力が出にくい理由を、屈折異常、不同視、斜視、器質的要因に分けて確認する必要があると分かった。

斜視・斜位

見る流れ：カバーテスト → プリズムカバーテスト → 眼球運動 → 両眼視

まとめ方：眼位のずれは量だけでなく、遠見・近見、方向、融像や抑制と合わせて見る必要があると分かった。

複視

見る流れ：単眼性/両眼性 → 眼位 → 眼球運動 → 自覚的検査 → 両眼視

まとめ方：複視では、どの方向で強くなるか、眼球運動制限があるかを合わせて確認することが大切だと分かった。

緑内障疑い

見る流れ：眼圧 → 眼底 → OCT → 視野

まとめ方：緑内障では、構造の変化と視野の変化が対応しているかを見る必要があると分かった。

学生用：自己評価

提出前に、自分の学習の変化を短く確認します。できなかったこと探しではなく、次に見ることを決めるための確認です。

今週できるようになったこと

最初より見えるようになったこと、説明できるようになったことを書きます。

まだ一緒に確認したいこと

自分だけでは判断しにくいことを、指導者に聞ける形にします。

来週、自分で先に見たいこと

指導を受ける前に、自分で意識して観察することを1つ決めます。

提出前に、学生自身で最低限確認します。すべて完璧である必要はありませんが、必須4項目が埋まっているかを見ます。

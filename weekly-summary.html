週のまとめの目的

注意：診断名を正解として当てる必要はありません。検査が何を確認するために行われているのかを、1週間の経験から説明できるようにすることが目的です。
書く量の目安：すべてを長く書く必要はありません。まずは必須4項目だけ書ければ十分です。余裕がある場合に、分かったこと・分からないこと・質問を足します。

1. 点をつなげる

1週間の場面を並べ、同じ課題や繰り返し出てきた所見を見つけます。

2. 検査の目的を言う

検査名だけでなく、何を確認するための検査かを書きます。

3. 来週へつなげる

来週見ること、実技で意識すること、調べ直すことを1〜2個に絞ります。

まずは必須4項目

週のまとめは、この4つが書ければ提出用の骨子になります。

必須1. 今週見た場面

症状・所見・検査場面を1つ選びます。広げすぎないことが大切です。

必須2. 関係しそうな原因

疾患・病態の候補を1〜2個に絞ります。確定ではなく、考える候補として書きます。

必須3. 必要な検査と見ること

原因を分けて考えるために、どの検査で何を見るかを書きます。

必須4. 来週確認したいこと

観察目標または実技目標を1〜2個に絞ります。

余裕があれば追加する項目

必須4項目が書けたあとに、必要に応じて追加します。

任意1. 今週分かったこと

単なる感想ではなく、検査と原因がどうつながったかを書きます。

任意2. まだ分からないこと

分からないことを、来週確認できる形にします。

任意3. 指導者に確認したいこと

質問は広げすぎず、今週見た場面に結びつけます。

本文にするときの短い型

型：今週見た場面 → 関係しそうな原因 → 必要な検査 → 来週確認したいこと

短い書き出し例

今週は、＿＿の場面を見た。この場面では、＿＿が関係しそうだと考えた。そのため、＿＿の検査で＿＿を確認する必要がある。来週は、＿＿と＿＿の結果がどうつながるかを見たい。

少し詳しく書く場合の型

今週は、＿＿の場面を複数回見た。最初は＿＿の検査手順に注目していたが、指導を受ける中で、＿＿を確認するために＿＿の検査が必要になることを学んだ。

特に、＿＿の検査では＿＿を見ることで、＿＿と＿＿を分けて考える手がかりになると分かった。来週は、＿＿の場面で、＿＿と＿＿の検査結果がどのようにつながっているかを意識して観察したい。

テーマ別の例

弱視疑い

見る流れ：視力 → 屈折 → 矯正視力 → 眼位 → 両眼視 → 眼底確認

まとめ方：視力が出にくい理由を、屈折異常、不同視、斜視、器質的要因に分けて確認する必要があると分かった。

斜視・斜位

見る流れ：カバーテスト → プリズムカバーテスト → 眼球運動 → 両眼視

まとめ方：眼位のずれは量だけでなく、遠見・近見、方向、融像や抑制と合わせて見る必要があると分かった。

複視

見る流れ：単眼性/両眼性 → 眼位 → 眼球運動 → 自覚的検査 → 両眼視

まとめ方：複視では、どの方向で強くなるか、眼球運動制限があるかを合わせて確認することが大切だと分かった。

緑内障疑い

見る流れ：眼圧 → 眼底 → OCT → 視野

まとめ方：緑内障では、構造の変化と視野の変化が対応しているかを見る必要があると分かった。

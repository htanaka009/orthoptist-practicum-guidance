週のまとめの目的

注意：診断名を当てることが目的ではありません。検査が何を確認するために行われているのかを、1週間の経験から説明できるようにすることが目的です。
毎日レポートとの違い：毎日は「今日の場面 → 明日見ること」。週のまとめは「複数日の場面 → 疾患・検査の関連 → 来週の課題」です。

1. 点をつなげる

月曜から金曜までの場面を並べ、同じ課題や繰り返し出てきた所見を見つけます。

2. 検査の目的を言う

検査名だけでなく、何を確認するための検査かを書きます。

3. 来週へつなげる

来週見ること、実技で意識すること、調べ直すことを1〜2個に絞ります。

週のまとめ：記入欄

1. 今週よく見た症状・所見

見えにくさ、眼位のずれ、複視、変視、視野異常、説明で迷った場面など。

2. 関係する疾患・病態

確定ではなく、考える候補として書きます。

3. 鑑別するために必要な検査

検査名だけでなく、その検査で何を分けたいかを書きます。

4. 各検査で見る所見

左右差、矯正で改善するか、偏位量、眼球運動、抑制、視野欠損、OCT所見など。

5. 今週分かったこと

単なる感想ではなく、検査と疾患・病態がどうつながったかを書きます。

6. まだ分からないこと

分からないことを、来週確認できる形にします。

7. 来週確認したいこと

観察目標または実技目標を1〜2個に絞ります。

8. 指導者に確認したいこと

質問は広げすぎず、今週見た場面に結びつけます。

本文にするときの型

型：今週見た場面 → 関係する疾患・病態 → 必要な検査 → 分かったこと → 来週確認したいこと

書き出し例

今週は、＿＿の場面を複数回見た。最初は＿＿の検査手順に注目していたが、指導を受ける中で、＿＿を確認するために＿＿の検査が必要になることを学んだ。

特に、＿＿の検査では＿＿を見ることで、＿＿と＿＿を分けて考える手がかりになると分かった。一方で、検査結果をどの順番で関連付けて考えるかはまだ十分に理解できていない。

来週は、＿＿の場面で、＿＿と＿＿の検査結果がどのようにつながっているかを意識して観察したい。

テーマ別の例

弱視疑い

見る流れ：視力 → 屈折 → 矯正視力 → 眼位 → 両眼視 → 眼底確認

まとめ方：視力が出にくい理由を、屈折異常、不同視、斜視、器質的要因に分けて確認する必要があると分かった。

斜視・斜位

見る流れ：カバーテスト → プリズムカバーテスト → 眼球運動 → 両眼視

まとめ方：眼位のずれは量だけでなく、遠見・近見、方向、融像や抑制と合わせて見る必要があると分かった。

複視

見る流れ：単眼性/両眼性 → 眼位 → 眼球運動 → 自覚的検査 → 両眼視

まとめ方：複視では、どの方向で強くなるか、眼球運動制限があるかを合わせて確認することが大切だと分かった。

緑内障疑い

見る流れ：眼圧 → 眼底 → OCT → 視野

まとめ方：緑内障では、構造の変化と視野の変化が対応しているかを見る必要があると分かった。

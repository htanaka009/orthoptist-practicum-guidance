このページの目的

注意：診断名を正解として当てる必要はありません。このページは、実習生が「その検査で何を確認しているのか」を説明できるようにするための補助資料です。診療上の判断は指導者・医師の判断に従います。
まず大事なこと：最初からすべての疾患や検査を挙げる必要はありません。今週見た場面から1つ選び、関係しそうな原因を1〜2個、必要な検査を1〜3個に絞ります。

学生用：基本の問い

「鑑別」という言葉は、ここでは原因を分けて考えることとして扱います。

問いの順番

  1. 今週見た場面は何か。
  2. 何が問題になっていたか。
  3. 関係しそうな原因は何か。
  4. その原因を分けて考えるために、どの検査が必要か。
  5. その検査で何を見ているか。

よい書き方

見えにくさの原因を考えるために、まず矯正視力と屈折を確認する。矯正しても視力が出にくい場合は、弱視や眼底疾患なども考え、眼位や眼底所見との関係を見る。

学生用：言葉に迷ったとき

中級編で出てくる言葉を、学生が書きやすい形に言い換えます。専門語を避けるのではなく、意味を確認してから使います。

器質的要因眼の構造や神経など、形や組織の変化が関係する可能性屈折だけでは説明しにくい原因も考える

学生用：調べる問いの作り方

今日・今週扱う調べものは、疾患名や検査名だけで終わらせず、場面と確認したいことを入れます。保留リストに入れる疑問は少し広くてもかまいません。

基本形：「＿＿の場面で、＿＿を確認するために、＿＿について調べる」または「＿＿の検査で、何を確認しているか」
広すぎる書き方絞った書き方
弱視について調べる視力が出にくい小児で、屈折異常と弱視をどう分けて考えるか
斜視について調べるカバーテストで、顕性偏位と潜伏偏位をどう見分けるか
視力検査について調べる視力検査で患者さんが迷ったとき、検査者が何を確認しているか
両眼視について調べる眼位のずれがあるとき、両眼視機能検査で何を確認するか
OCTについて調べる視力が出にくいとき、OCTで黄斑部の形態をどう確認するか

1. 疾患名だけで終わらせない

「弱視」だけでなく、どの場面で考えたのかを書きます。

2. 検査名だけで終わらせない

「OCT」だけでなく、その検査で何を見るのかを書きます。

3. 場面を入れる

今日または今週見た場面とつなげます。

4. 確認したいことを入れる

何を分けて考えたいのかを明確にします。

5. 1項目1問い

1つの項目に複数の疑問を詰め込みません。

学生用：疾患×検査カード

週のまとめや調べ学習では、疾患名だけでなく「必要な検査」と「その検査で見ること」をセットで整理します。

記入テンプレート

  1. テーマ：例：見えにくい、眼位がずれる、複視、変視、視野が欠ける
  2. 今週見た場面：実習で見た場面を1つ
  3. 関係しそうな原因：疾患・病態の候補を1〜2個
  4. 必要な検査：検査名だけでなく、目的を書く
  5. 検査で見ること：左右差、偏位量、抑制、構造変化など
  6. 次に見ること：次回どの結果同士を関連付けて見るか

記入例：眼位がずれる場面

整理メモ

今週見た場面：
カバーテストで、検査者が固視と眼球の動きを確認していた。

関係しそうな原因：
斜視、斜位、間欠性斜視。

次に見ること：
眼位検査の結果と、両眼視機能の結果がどうつながるかを見る。

検査をつなげる一文

カバーテストでは、眼位のずれがあるかを確認する。プリズムカバーテストでは、そのずれがどのくらいかを測る。両眼視機能検査では、そのずれが見え方にどう影響しているかを確認する。

学生用：まず見る順番カード

中級編では、すべてを一度に考えません。よくある場面ごとに、まず何を見るかを順番で確認します。

見えにくいとき

  1. 屈折で説明できるか
  2. 矯正しても視力が出にくいか
  3. 左右差があるか
  4. 眼位や眼底所見と合うか

一文例：見えにくさを考えるときは、まず屈折と矯正視力を確認し、必要に応じて眼位や眼底所見と合わせて見る。

眼位がずれるとき

  1. 顕性偏位か潜伏偏位か
  2. 遠見・近見で変わるか
  3. ずれの量と方向はどうか
  4. 両眼視に影響しているか

一文例：眼位がずれる場面では、ずれの有無だけでなく、量、方向、両眼視への影響を合わせて見る。

複視があるとき

  1. 単眼性か両眼性か
  2. どの方向で強くなるか
  3. 眼球運動制限があるか
  4. 眼位検査や自覚的検査と合うか

一文例：複視では、出る方向と眼球運動の状態を合わせて、原因の手がかりを整理する。

中心が見にくい・ゆがむとき

  1. 中心視力に影響しているか
  2. 左右差があるか
  3. 変視があるか
  4. OCTや眼底所見と合うか

一文例：中心が見にくい場合は、視力だけでなく、変視や黄斑部の形態と合わせて見る。

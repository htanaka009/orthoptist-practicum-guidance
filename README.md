# 視能訓練士実習｜明日の実習につなげる整理ナビ

指導者が、実習生と一緒に帰宅前整理・レポート骨子作成・提出レポート添削を進め、翌日の観察や実技につなげるためのHTML簡易ナビです。

## 主な流れ

帰宅前整理 → レポート骨子 → 作成例で確認 → 提出 → 添削 → 添削後に次へ戻す、という循環で使います。

## 注意

- 患者情報、学生名、学校名、施設名など、特定につながる情報は掲載しないでください。
- 本資料は、指導者が実習生と一緒に振り返りを進めるための補助資料です。
- 本資料は成績評価や診療上の判断を自動化するものではありません。
- GitHub Pagesで公開する場合、この資料はインターネット上に公開されます。

## 公開方法

このリポジトリのルートに `index.html` を置き、GitHub Pages を `main` ブランチの root から公開してください。

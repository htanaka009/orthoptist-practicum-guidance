# 視能訓練士実習｜明日の実習につなげる整理ナビ

指導者が、実習生と一緒に毎日の実習整理、レポート骨子作成、検査と疾患の関連付け、週のまとめ、提出レポート添削を進め、次の観察や実技につなげるためのHTML簡易ナビです。

## 主な流れ

初級編で毎日の場面を整理し、中級編で週単位の学習につなげます。

帰宅前整理 → レポート骨子 → 作成例で確認 → 疾患と検査をつなげる → 週のまとめ → 提出 → 添削 → 添削後に次へ戻す、という循環で使います。

## 初級編と中級編

### 初級編：毎日の整理

- 使うタイミング：毎日の帰宅前、または日々の提出レポート作成前。
- 目的：その日に見た場面を1つ選び、気づき、考えたこと、明日見ることにつなげる。
- 到達目標：学生が「今日見た場面」「考えたこと」「明日見ること」を自分の言葉で書ける。
- 次へ進む目安：毎日のレポートで、場面と明日見ることが安定して書けるようになったら、中級編を使う。

### 中級編：検査と疾患をつなげる

- 使うタイミング：週末、実習中盤以降、または同じ検査・所見を複数回見たあと。
- 目的：今週見た場面から、関係しそうな原因、必要な検査、検査で見ることをつなげる。
- 到達目標：学生が「なぜその検査が必要だったのか」「その検査で何を確認していたのか」を説明できる。
- 注意：疾患名を正解として当てることが目的ではなく、検査の目的を言語化することを優先する。

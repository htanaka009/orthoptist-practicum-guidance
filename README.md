# 視能訓練士実習 初級編｜紙提出用 振り返り資料 v5

## 内容
- 学生用：日々の振り返りシート
  - 冒頭に「今日の提出文の完成イメージ」を追加
- 学生用：日々のレポート例カード
- 週まとめ用：共通点を見つけて週まとめへ
  - 共通点チェック欄
  - 共通点の種類を示す表
  - 共通点に名前をつける欄
  - 共通点が見つからないときの支援
  - 共通点から本文にする型
  - 調べることリストを週まとめに使う欄
- 指導者用：帰宅前・週まとめ声かけガイド
  - 調べることが増えすぎたときの声かけ
- 一括印刷版

## フォント
HTML/CSSでは `font-family: "Noto Sans JP"` を指定しています。
印刷するPCに Noto Sans JP が入っていれば、そのフォントで印刷されます。
フォントファイルは同梱していません。

## 記載方針
年齢、性別、主訴、疾患・疑い、屈折値、視力、検査に関係する情報は記載可としています。
氏名、患者ID、住所、電話番号、具体的な来院日などは書かない前提です。

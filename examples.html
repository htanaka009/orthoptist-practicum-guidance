レポート作成例

初級編では、帰宅前整理で出した内容がレポート骨子と本文へ変わる流れを確認します。中級編では、初級編の場面をもとに、検査の目的や関係しそうな原因へ広げます。

ホーム初級編中級編指導運用

このページでやること

1. 初級例を見る

場面・気づき・考えたこと・明日見ることの形を確認します。

2. 骨子を見る

整理メモがレポート骨子になる流れを見ます。

3. 本文を見る

骨子を文章にした例を確認します。

4. 中級へ発展

検査の目的や関係しそうな原因を足す例を確認します。

使い方

注意：この例文は、そのまま写すためのものではありません。自分が見た場面、自分が気になったこと、自分が明日または来週見ることに置き換えて使います。
見るポイント：初級編は「場面 → 気づき → 考えたこと → 明日見ること」。中級編は「場面 → 関係しそうな原因 → 必要な検査 → 検査で見ること → 来週確認したいこと」です。

例1：観察中心のレポート例

整理メモ

今日見た場面：
視力検査で、患者さんが答えに迷っていた場面。

気になったこと：
検査者がすぐに促さず、少し待っていたこと。

関係する知識：
患者さんの反応を見ながら、答えを急がせすぎないことが大切。

今日考えたこと：
反応を見ながら待つことも検査の一部だと考えた。

明日見ること：
迷ったときの声かけのタイミングを見る。

レポート骨子

  1. 視力検査で患者さんが答えに迷う場面を見た。
  2. 検査者が少し待っていたことが気になった。
  3. 患者さんの反応を見ながら進めることが大切である。
  4. 待つことも検査の一部だと考えた。
  5. 明日は声かけのタイミングを見る。

レポート例

今日の実習では、視力検査で患者さんが答えに迷っている場面を見た。検査者はすぐに答えを促すのではなく、患者さんの反応を見ながら少し待っていた。

この場面から、検査では手順を進めるだけでなく、患者さんの反応を見ながら待つことや声をかけることも重要だと考えた。明日は、患者さんが迷ったときに検査者がどのタイミングで声をかけているかを見たい。

例2：実技中心のレポート例

整理メモ

今日見た場面：
自分が視力検査の説明をした場面。

気になったこと：
最初に何をどの順番で説明すればよいか迷った。

関係する知識：
検査前の説明では、方法、答え方、分からないときの対応を伝える必要がある。

今日考えたこと：
説明が不十分だと、患者さんが不安になったり答え方に迷ったりする可能性がある。

明日見ること：
検査者が何をどの順番で説明しているかを見る。

レポート骨子

  1. 視力検査の前に説明を行った。
  2. 説明の順番に迷った。
  3. 検査方法、答え方、分からないときの対応を伝える必要がある。
  4. 説明は検査結果にも影響すると考えた。
  5. 明日は説明の順番を観察する。

レポート例

今日の実習では、視力検査の前に患者さんへ説明を行った。実際に説明しようとすると、最初に何を伝えるべきか、どの順番で話せばよいか迷った。

検査前の説明では、検査方法だけでなく、答え方や分からないときの対応を分かりやすく伝える必要がある。明日は、検査者が検査前に何をどの順番で説明しているかを観察し、自分が実施するときには説明内容を整理してから始めたい。

例3：眼位検査のレポート例

整理メモ

今日見た場面：
カバーテストで、検査者が患者さんの固視と眼球の動きを確認していた場面。

気になったこと：
カバーを外した直後の小さな動きを見逃さないように観察していたこと。

関係する知識：
眼位検査では、固視状態、カバーのタイミング、眼球運動の方向を合わせて確認する必要がある。

今日考えたこと：
眼球の動きだけを見るのではなく、患者さんがどこを見ているかを確認しながら観察することが大切だと考えた。

明日見ること：
検査者が固視をどのように確認し、どのタイミングで眼球の動きを見ているかを見る。

レポート骨子

  1. カバーテストで固視と眼球の動きを確認する場面を見た。
  2. カバーを外した直後の小さな動きに注目していたことが気になった。
  3. 眼位検査では、固視、カバーのタイミング、動きの方向を合わせて見る必要がある。
  4. 患者さんの固視を確認しながら観察することが大切だと考えた。
  5. 明日は固視確認と眼球運動を見るタイミングを観察する。

レポート例

今日の実習では、カバーテストで検査者が患者さんの固視と眼球の動きを確認している場面を見た。特に、カバーを外した直後の小さな動きを見逃さないように観察していたことが印象に残った。

眼位検査では、眼球の動きだけでなく、患者さんがどこを見ているか、カバーのタイミングが適切かを合わせて確認する必要があると考えた。明日は、検査者が固視をどのように確認し、どのタイミングで眼球の動きを見ているかを意識して観察したい。

例4：知識まとめから実習場面へつなげる例

弱くなりやすい例

視力検査は視機能を評価する重要な検査である。正確に測定することが大切である。

間違いではありませんが、今日の場面がないため、実習レポートとしては弱くなります。

整理し直す

知識：
視力検査は正確な測定が大切。

今日の場面：
患者さんが答えに迷ったとき、検査者がすぐに次へ進まず確認していた。

考えたこと：
正確な測定のためには、手順を守るだけでなく、患者さんの反応を確認しながら進めることが必要だと考えた。

レポート例

視力検査は視機能を評価するための基本的な検査であり、正確な測定が重要である。今日の実習では、患者さんが答えに迷ったとき、検査者がすぐに次へ進まず、患者さんの反応を確認しながら検査を進めている場面を見た。

この場面から、正確な測定のためには、検査手順を理解するだけでなく、患者さんの反応を観察しながら進めることが必要だと考えた。明日は、患者さんが迷ったときに、検査者がどのように確認しているかを見たい。

例5：初級例から中級例へ発展させる例

目的：初級編の「今日見た場面」を、週のまとめで「必要な検査と見ること」まで広げます。診断名を当てるのではなく、検査の目的を説明できる形にします。

初級編で書けていること

場面：
視力検査で、患者さんが答えに迷っていた。

考えたこと：
患者さんの反応を見ながら待つことも検査の一部だと考えた。

明日見ること：
迷ったときの声かけのタイミングを見る。

中級編で足す視点

関係しそうな原因：
屈折異常、弱視、眼位の影響、器質的要因など。

必要な検査：
矯正視力、屈折検査、眼位検査、必要に応じた眼底・OCTなど。

検査で見ること：
矯正で改善するか、左右差があるか、眼位のずれが見え方に影響していないか。

週のまとめ用の骨子

  1. 今週は、視力が出にくい場面や、答えに迷う場面を複数回見た。
  2. 見えにくさには、屈折異常、弱視、眼位の影響、器質的要因などが関係する可能性がある。
  3. そのため、視力検査だけでなく、屈折検査、矯正視力、眼位検査などの結果を合わせて見る必要がある。
  4. 来週は、視力が出にくい場面で、屈折・矯正視力・眼位の結果がどのようにつながっているかを見る。

中級レベルの本文例

今週は、視力検査で患者さんが答えに迷う場面や、矯正しても視力が出にくい場面を見た。初めは声かけや待ち方に注目していたが、視力が出にくい理由を考えるには、視力検査だけでなく、屈折検査や矯正視力、眼位検査の結果も合わせて見る必要があると分かった。

見えにくさには、屈折異常、弱視、眼位の影響、器質的要因などが関係する可能性がある。そのため、検査ごとの結果を別々に見るのではなく、矯正で改善するか、左右差があるか、眼位のずれが見え方に影響していないかを関連付けて考えることが大切だと感じた。来週は、視力が出にくい患者さんで、屈折・矯正視力・眼位の結果がどのようにつながっているかを意識して観察したい。

指導者向け：初級編から中級編へ進めるときは、学生に疾患名を多く挙げさせるより、「どの検査で何を確認していたか」を1〜3個に絞って言わせる方が使いやすくなります。

例6：眼位検査を中級レベルへ発展させる例

初級編で書けていること

場面：
カバーテストで、固視と眼球の動きを確認していた。

考えたこと：
眼球の動きだけでなく、患者さんがどこを見ているかを確認しながら観察することが大切だと考えた。

明日見ること：
固視確認と眼球運動を見るタイミングを観察する。

中級編で足す視点

関係しそうな原因：
斜視、斜位、間欠性斜視など。

必要な検査：
カバーテスト、プリズムカバーテスト、眼球運動、両眼視機能検査。

検査で見ること：
ずれの有無、ずれの量、方向、融像や抑制の状態。

中級レベルの本文例

今週は、カバーテストで固視と眼球の動きを確認している場面を見た。初めはカバーを外した直後の眼球の動きに注目していたが、眼位検査では、ずれがあるかだけでなく、その量や方向、両眼視への影響も合わせて確認する必要があると分かった。

カバーテストでは眼位のずれがあるかを確認し、プリズムカバーテストではずれの量を測る。さらに、両眼視機能検査では、そのずれが見え方にどう影響しているかを確認する。来週は、眼位検査の結果と両眼視機能の結果がどのようにつながっているかを意識して観察したい。
